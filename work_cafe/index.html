Web site

Cafe「Goo day coffee」（架空）

サイトURL : https://river10nao.github.io/cafe_Website/（PC画面のみ対応）

◼️概要
職業訓練校の演習課題
スペシャルティコーヒー専門カフェ（架空）
◼️使用アプリ
Figma / Photoshop / Illustrator
VS-code
（HTML / CSS / JavaScript）
◼️制作時間
デザイン : 1週間
コーディング : 3週間
◼️カラー
事前情報
1_ 店名 : Good Day Coffee
2_ コーヒー豆販売とカフェの営業
3_ 店舗数 : 神戸（本店）/ 大阪 / 京都の3店舗
4_ 訴求ポイント : お店で自家焙煎したコーヒー豆を販売 / 珈琲豆はスペシャルティーコーヒーのみを使用 / バリスタチャンピオンシップ優勝者が在籍
目的
他のカフェとの差別化を図り、お客様に興味を持っていただき、店舗への来店を促す。
コンセプト
・珈琲豆へのこだわりを伝える
・珈琲好きに響かせる
・店内の大人向けの落ち着いた雰囲気や居心地の良さをアピール
ペルソナ
30代男性 / 神戸在住 / 趣味はキャンプ / 毎朝豆から挽いて飲むほど珈琲好き / 何に対してもこだわりが強い / 最近スペシャルティコーヒーについて知り興味を持つ / 珈琲豆はいつもKALDIで購入している
ポイント
・トップページに大きく店内の様子を出し、落ち着いた雰囲気でくつろげる空間をイメージできるようにしました。
・全体的にふわっと下から文字を出す動きをつけ、柔らかい印象を与えます。
・オンラインで珈琲豆を販売するのもいいのではないかと思い、フッター部分にバナーを挿入してみました。
・Coffeeページのトップには、珈琲豆から珈琲になるまでの自作の動画を画面いっぱいに写し、目を引くようにしました。
・Menuページのトップでは、メニューの写真をスライダーで沢山見せ、種類豊富なのメニューに興味をもってもらえるようにしました。
