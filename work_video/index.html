◼️概要
職業訓練校での演習課題
神戸の旧居留地/南京町のPR動画
2分〜3分程度にまとめる
◼️使用アプリ
Premiere Pro
◼️制作時間
企画 : 2日
撮影 : 1日
編集 : 2日
◼️素材
自身で撮影した動画や写真
目的
神戸（旧居留地/南京町）の魅力をたっぷり伝え、観光へ訪れてもらう。
企画・撮影の流れ
１、事前調査 : 旧居留地と南京町はどのような魅力のある街なのかを調査し、アピールポイントを考える。
２、撮影箇所選定 : 観光名所やおすすめしたい場所を選定する。
３、動画構成 : 絵コンテを書き、動画の大体の流れを決める。
４、撮影準備 : 施設の開場時間を調べたり、回りやすいルートを考えておく。YouTube等で動画の撮り方を勉強をしておく。
５、撮影 : 自身のiPhoneで撮影。手ブレに注意し、タイムラプスやカメラトランジションを活用し工夫しながら撮影。
ターゲット
歴史好き、一人旅が好きな人、神戸に観光を考えている人 / 20代〜30代男女
コンセプト
”現代×歴史×文化”の旅がコンセプトです。 前半→旧居留地、後半→南京町と構成を分けました。旧居留地では明治時代から伝わった美しい西洋の建物や街並み、南京町では中国系移民の町だったという名残を感じられる活気ある中華街をアピール。現代と融合させて、異国情緒溢れる神戸を伝えられるように制作しました。

wedding venue

URL : https://youtu.be/fupJpVWa5rU?si=GCDEr9OfkyfNnU5i

◼️概要
職業訓練校での演習課題
結婚式場のPR動画
1分以内にまとめる
◼️使用アプリ
Premiere Pro
◼️制作時間
企画 : 1日
編集 : 2日
◼️提供素材
講師からいただいたもの
花嫁や式場のフリー動画
目的
結婚式場の魅力を最大限に伝え、見た人に「ここで結婚式を挙げたい」と強く感じてもらうことです。美しいロケーションと贅沢な非日常感を視覚的にアピールし、人生において最も大切な一日を、心に深く刻まれる特別な一日にするための理想的な場所であることを訴求します。
商品の訴求ポイント
１、開放的な空間：広々とした開放感のある式場で、ゲストもリラックスして過ごせます。
２、美しいロケーション：雄大な自然に囲まれた場所で、非日常的で贅沢な雰囲気を提供します。
３、映画のような結婚式：まるで映画のワンシーンのようなドラマチックな結婚式が実現可能です。
４、特別感の演出：通常の式場では味わえない、特別な一日を提供します。
５、オープンな気持ち：自然に包まれた環境で、ゲストも新郎新婦も気持ちがオープンになり、素直な感謝や思いを伝え合える場を作ります。
ターゲット
20代〜30代男女 結婚を考えている人、式場を探している人
コンセプト
このPR動画は、「映画のワンシーンのような非日常的で贅沢な結婚式」をコンセプトに制作しました。開放的で美しいロケーションを強調し、視覚的なインパクトと感動を与えることを目指しました。動画には、自然に包まれた雄大な風景と共に、新郎新婦やゲストがリラックスし、心からの感謝や思いを素直に伝え合う様子を盛り込みました。特別な一日が誰の心にも鮮明に残るような素敵な式が叶う、というメッセージを込めて制作しました。
